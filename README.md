# KYC Verification App

This project extracts KYC details (Name, DOB, ID Number) from document images using AI.

## Tech Stack
- Python
- Flask (Backend)
- Streamlit (Frontend)
- Google Gemini API

## Features
- Upload KYC document
- Extract Name, DOB, ID
- Validate and store data

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend:
python app.py

3. Run frontend:
streamlit run frontend.py
